# TCP & UDP

TCP 和 UDP 是传输层的两个主要协议



## TCP(Transmission Control Protocol)

TCP 是面向链接的、可靠的流协议。

- 顺序控制 / 重发控制
- 流量控制
- 拥塞控制
- 提高网络利用率

### 三次握手

1. 发送方 A 询问接收方 B 是否能收到 A 的请求

2. B 确认，并询问 A 是否能收到 B 的响应

3. A 确定（此次允许携带数据）

为什么不可以两次握手？如果因网络问题导致 A 连接请求重试了多次，此时如果 B 确认即建立连接，将导致有多个连接被建立而浪费资源。



### 四次挥手

1. A：我发完咯
2. B：知道了，你再等我一下，我处理下数据
3. B：我数据处理完了
4. A：好的，断开吧
